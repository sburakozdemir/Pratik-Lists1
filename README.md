# Pratik-Lists1

Bu proje, Patika Plus gala gecesi için bir ziyaretçi listesi oluşturan ve bu listedeki isimleri ekrana yazdıran basit bir C# konsol uygulamasıdır.

## Proje Açıklaması

Bu uygulama, aşağıdaki işlevleri gerçekleştirir:

1. Önceden belirlenmiş bir ziyaretçi listesi oluşturur.
2. Bu listedeki isimleri numaralandırarak ekrana yazdırır.
